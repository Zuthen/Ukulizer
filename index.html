Guitar to uke tab converter


How to use that tool? It's simple!

  1. Paste a guitar tab into field below
  2. You can change the number of frets on the neck if it is different from 18
  3. Click Convert to ukulele tab button
  4. If everything is fine result should appear below.
  5. If your tab consists of multiple lines just repeat the process for each line. They will appear will appear one under the other
  6. If You want to export result to pdf file just click on Download pdf button (You can set the song title and/or artist name in the corresponding field)

ℹ️ For best results

  • make sure that each string notations starts with exactly one letter and one "|" sign
    • For example string like D|-12-10--| will work properly but D4-||------| won't
  • make sure that each string notations ends with "|" sign. All other endings may affect the rhythmics
    • For example string like e|-12-12-12-| will work properly but D|------|- won't
  • avoid x2 notation. Remove "x2", type the line to the input field and click convert button twice
